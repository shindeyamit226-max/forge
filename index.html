Features

Everything a coding tool should be

Built from the ground up to be the best. Not a wrapper around an API — a real tool with real intelligence.

🧠

Knowledge Graph

Like Obsidian, but for code. Every function, class, and file is connected. Change one thing — instantly know what breaks.

🔄

Self-Correcting Agent

Plans, executes, verifies, and fixes its own mistakes. If tests fail, it analyzes the error and retries with the fix.

📝

20 Language Parsers

Real AST parsing for Python, JS, TS, Go, Rust, Java, C++, Ruby, PHP, Swift, Kotlin, Scala, SQL, and more.

🔍

Semantic Search

Find code by meaning, not keywords. BM25 ranking over TF-IDF index. Ask "how does auth work?" and get the answer.

🛡️

Security Scanner

Detects SQL injection, XSS, hardcoded secrets, weak crypto, and 10+ vulnerability patterns. Every issue has a fix.

🔌

Plugin System

Drop a .py file in ~/.forge/plugins/ — it auto-loads. Add tools, hooks, and generators in seconds.

How It Works

Think. Act. Verify. Learn.

The ReAct pattern — the same approach used by the most advanced AI agents.

1

🧠 Understand

Forge reads your codebase, builds a knowledge graph, and understands the relationships between every symbol.

2

📊 Analyze

Before making any change, it runs impact analysis. What calls this? What tests cover it? What's the risk level?

3

📋 Plan

Complex tasks are broken into ordered steps. Each step is specific, actionable, and verifiable.

4

⚡ Execute

Surgical edits using AST-aware tools. Not text replacement — real code understanding.

5

✅ Verify

Runs tests, checks for errors, validates the change. If something breaks, it self-corrects.

6

🧠 Learn

Remembers your preferences, coding patterns, and solutions. Gets better every session.

Install

Up and running in 60 seconds

No accounts. No API keys. No cloud. Just code.

1

Clone & Install

git clone https://github.com/shindeyamit226-max/forge.git
cd forge
pip install -e .
2

Pull a Model

# Install Ollama: https://ollama.com
ollama pull codellama:13b
3

Run Forge

# Interactive mode
forge

# One-shot mode
forge run "add authentication to the API"
